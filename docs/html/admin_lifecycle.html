GUARDS LAYER
Auth & Authorization
AuthGuard RolesGuard
@UseGuards(AuthGuard, RolesGuard)
@Roles(UserRole.ADMIN)

1. **AuthGuard**: Xác thực JWT token, lấy thông tin giải mã đưa vào `request.user`.
2. **RolesGuard**: Đọc metadata `@Roles` kiểm tra vai trò người dùng có phải `ADMIN` không.

EXCEPTION FILTER GATE
HttpExceptionFilter

Bắt giữ tất cả các Exception xảy ra trong quá trình xử lý request, chuyển thành payload lỗi chuẩn hóa.

{
"statusCode": 401 / 403 / 400,
"message": "Error details message",
"timestamp": "2026-05-21..."
}
VALIDATION PIPE
class-validator / DTO

Kiểm tra tính hợp lệ dữ liệu Category:

name: string @IsString() @IsNotEmpty() @MinLength(3)
slug: string @IsString() @IsNotEmpty() @Matches(/^[a-z0-9-]+$/)

Kiểm tra các thuộc tính đầu vào của Product:

name: string | slug: string @IsString() @IsNotEmpty()
price: string (Decimal mapping) @IsString() @IsNotEmpty() @IsDecimal()
stock: number @IsInt() @Min(0)
categoryId: number @IsInt() @IsNotEmpty()

Xác thực tham số truy vấn & cập nhật đơn hàng:

status: OrderStatus @IsOptional() / @IsEnum(OrderStatus)
page: number | limit: number @IsOptional() @Type(() => Number) @IsInt()
RESPONSE FORMATTER
api-response.util.ts

Đóng gói kết quả thành công theo mẫu JSON phản hồi thống nhất của hệ thống.

success(category)
// Returns: { success: true, data: category }
success(product)
// Returns: { success: true, data: product }
paginated(data, total, page, limit)
// Returns: { success: true, data, meta }
ADMIN CONTROLLER
admin.controller.ts

Định tuyến request và phân phối dữ liệu tham số vào các phương thức xử lý nghiệp vụ.

@Post('categories')
createCategory(@Body() dto: CreateCategoryDto)

@Patch('categories/:id')
updateCategory(@Param('id') id, @Body() dto)
@Post('products')
createProduct(@Body() dto: CreateProductDto)

@Patch('products/:id')
updateProduct(@Param('id') id, @Body() dto)

@Delete('products/:id')
softDeleteProduct(@Param('id') id)
@Get('orders')
getOrders(@Query() query: AdminOrderQueryDto)

@Patch('orders/:id/status')
updateOrderStatus(@Param('id') id, @Body() dto)

Converts Prisma.Decimal to Number
